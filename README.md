# ppt-paperlogy-inspired-skill

마크다운 문서 하나를 통일된 디자인의 **편집 가능한 PPT**로 만들어 주는 Claude Code 스킬.

## 이것만 하시면 됩니다
1. `decks/` 안에 폴더를 하나 만든다 - 예: `decks/01-내주제/`
2. 그 폴더에 PPT로 만들고 싶은 내용을 넣는다 - 마크다운이 가장 좋고, 다른 파일이어도 됩니다. (이미지가 있으면 같이)
3. Claude에게: **"이걸로 PPT 만들어줘"**

→ 같은 폴더에 `.pptx`가 생깁니다. 차트·표·도형 전부 파워포인트에서 그대로 편집돼요.

내용이 부족하거나 애매하면 Claude가 물어보며 함께 다듬어 갑니다.

## 바꾸고 싶으면
- **로고** — `assets/your-logo.png`를 본인 것으로 교체
- **강조색** — Claude에게 "강조색 ○○로 바꿔줘"

나머지(배경·폰트·레이아웃)는 통일성을 위해 고정되어 있으나, Claude와 함께 대화 나누며 개선해 나갈 수 있습니다.

## 원칙
원문에 없는 사실·수치·출처는 만들어내지 않습니다. 차트·표는 원문에 실제로 있는 데이터로만 그립니다.
