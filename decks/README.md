# decks/

주제(프레젠테이션)마다 **폴더 하나**를 만든다. 입력(md·이미지)과 출력(pptx)이 한 폴더에 모인다.

## 규칙
- 폴더명: `NN-슬러그` — `NN`은 2자리 인덱스(기존 최대 +1), 슬러그는 짧은 주제명.
  - 예: `01-두쫀쿠`, `02-반도체시장`, `03-q3-실적`
- 폴더 안에:
  - `<주제>.md` — 원문 마크다운 (필수)
  - `images/` 또는 이미지 파일들 — md에서 `![](상대경로)`로 참조 (선택)
  - `spec.json` — Claude가 생성하는 슬라이드 스펙 (중간 산출물)
  - `<주제>.pptx` — 최종 결과물

## 예시
```
decks/
└── 01-두쫀쿠/
    ├── 두쫀쿠.md
    ├── images/
    │   └── chart.png
    ├── spec.json
    └── 두쫀쿠.pptx
```

빌드: `python scripts/build.py decks/01-두쫀쿠/spec.json --out decks/01-두쫀쿠/두쫀쿠.pptx`
